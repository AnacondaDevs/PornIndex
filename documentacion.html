PornIndex

Proyecto de catálogo y gestión de actrices con múltiples enlaces y filtros personalizados.

Ver repositorio en GitHub

Funcionalidades principales

  • Listado y búsqueda de actrices con filtros por nombre, vistas y favoritas.
  • Soporte para múltiples enlaces (Pornhub, Eporner, Instagram, OnlyFans, etc.) por actriz.
  • Modo de visualización en lista o galería.
  • Agregar, editar y eliminar actrices con formulario dinámico.
  • Marcado de actrices favoritas y estadísticas de vistas.
  • Exportar e importar la base de datos en formato JSON.
  • Modo oscuro, claro y personalizado para la interfaz.

Cómo usar

Para ejecutar PornIndex en tu máquina local, sigue estos pasos:

  1. Clona el repositorio: git clone https://github.com/AnacondaDevs/PornIndex.git
  2. Entra en la carpeta del proyecto: cd PornIndex
  3. Instala las dependencias: npm install
  4. Ejecuta la aplicación: npm start
  5. Abre tu navegador en http://localhost:3000

¡Listo! Ya puedes comenzar a explorar, agregar y administrar actrices.

Componentes principales

A continuación un resumen de los componentes clave en el proyecto:

  • App.jsx: Componente principal que administra el estado global, filtros, y controla la visualización.
  • ActressCard.jsx: Tarjeta para mostrar la información resumida de cada actriz y los múltiples enlaces.
  • Modal.jsx: Componente reutilizable para mostrar formularios y detalles en ventanas modales.
  • AddActressForm.jsx: Formulario para agregar o editar actrices.
  • FeaturedActress.jsx: Destaca la actriz más vista.
  • Stats.jsx: Estadísticas generales del catálogo.
  • PendingModal.jsx: Modal para mostrar tareas pendientes o notas.

Extensiones y tips

  • Incorporar react-router para navegación a páginas de detalle dedicadas.
  • Agregar paginación para listas largas de actrices.
  • Implementar autenticación para usuarios y manejo de perfiles.
  • Mejorar diseño visual con frameworks CSS (Tailwind, Chakra UI, etc.) o Styled Components.
  • Soporte para imágenes locales o subida de imágenes desde el usuario.
